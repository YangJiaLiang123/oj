河北师大OJ平台Q&A

河北师大OJ平台能做什么

就如其他OJ平台一样，河北师大OJ平台旨在为河北师范大学软件学院和数信学院提供更见方便和具有针对性的算法学校平台，营造专业的竞赛气氛，并且为大家提供各大竞赛的参加捷径。

你可以使用OJ平台来学习算法知识，解决算法题目；参加或举办算法竞赛，一般校内代表队会由参加竞赛的优秀队伍中选出；使用讨论版和博客，记录或查看其他同学的学习经验；在资源模块下载软件、电子书或其他所需资料。

支持语言？

目前仅支持C和C++

代码提交注意事项

C和C++的main函数定义为"int main(void)"返回值为"return 0;"，因为判题核心还在开发阶段，尚不知违反定义规则会发生的怎样的错误。

算法的提交代码结果

编译异常：Compile Error

错误答案：Wrong Answer

正确：Accepted

超出时间限制：Time Limit Exceeded

超内存内存限制：Memory Limit Exceeded

运行时异常：Runtime Error

系统异常：System Error
